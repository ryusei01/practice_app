## トラブルシューティング用コマンド集

### データベースリセット
```bash

```

### キャッシュクリア
```bash
# フロントエンド
cd frontend
npx expo start -c

# バックエンド
cd backend
rm -rf __pycache__
```

### 依存関係再インストール
```bash
# バックエンド
pip install -r requirements.txt --force-reinstall

# フロントエンド
rm -rf node_modules package-lock.json
npm install
```
